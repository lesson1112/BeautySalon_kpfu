ИНСТРУКЦИЯ ПО ПРОГРАММЕ
«Салон красоты» (консольное приложение C#)

Откройте проект в Visual Studio 2022.

Запустите программу (Ctrl + F5).

Появится главное меню.

Введите номер пункта и нажмите Enter.

ПУНКТ 1: СПИСОК УСЛУГ

Показывает таблицу услуг салона:

ID услуги

Название

Цена

Длительность (в минутах)

Если цена указана некорректно (меньше 0) — услуга не создается.

ПУНКТ 2: РЕГИСТРАЦИЯ КЛИЕНТА

Шаг 1: Введите имя клиента
Шаг 2: Введите номер телефона

Клиент будет добавлен в систему.

Если клиент уже существует — программа уведомит об этом.

ПУНКТ 3: ПОИСК КЛИЕНТА

Шаг 1: Введите имя или телефон

Программа отобразит:

Информацию о клиенте

Все его записи

Общую сумму потраченных средств

Если клиент не найден — появится соответствующее сообщение.

ПУНКТ 4: ЗАПИСЬ НА УСЛУГУ

Шаг 1: Выберите клиента
Шаг 2: Выберите услугу по ID
Шаг 3: Выберите мастера
Шаг 4: Введите дату и время записи

Если мастер свободен — запись будет создана.

Если мастер занят или не работает в это время — программа сообщит об ошибке.

Стоимость услуги будет автоматически добавлена в общий расчет выручки.

ПУНКТ 5: ИНФОРМАЦИЯ О МАСТЕРАХ

Показывает:

Имя мастера

Специализацию

Расписание работы

Общий заработок

Заработок рассчитывается автоматически на основе оказанных услуг.

ПУНКТ 6: СТАТИСТИКА САЛОНА

Показывает:

Общее количество клиентов

Общее количество записей

Общую выручку салона

Все данные обновляются после каждой записи.

ПУНКТ 7: ВЫХОД

Закрывает программу.

ВАЖНО

Номер телефона можно вводить в любом формате

Повторная регистрация одного и того же клиента не допускается

Если мастер недоступен — запись не создается

При ошибке ввода программа возвращается в главное меню

Все расчеты выполняются автоматически

Программа предназначена для администраторов салона красоты.

Позволяет вести учет клиентов, услуг, мастеров и выручки.
